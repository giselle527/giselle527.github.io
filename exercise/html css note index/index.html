小知识补充

font-size字体大小、text-align文本对齐、charset页面字符集
常用的charset编码格式有2种，一种是utf-8，国际通用；一种是gb2312，国内最常用
边框宽：border-width:thin,medium,thick
边框线型：border-style:none,dotted,dashed,solid,double,groove,ridge,inset,outset
下划线：text-decoration:underline,none
删除线标签：s标签和del标签
上标标签和下标标签：sup标签和sub标签
斜体标签：i标签和em标签
加粗标签：b标签和strong标签
空标签又叫void element空位元素又叫自闭合标签

合并表格的行和列：
给td加属性rowspan="合并的行数"：要删除相应的被合并的行的td
给td加属性colspan="合并的列数"：要删除相应的被合并的列，即同一行的td

<!doctype html>就是html5的声明
注意<!doctype html>是声明，是放在<html>上面的，而不是来代替<html>的
<meta charset="utf-8">要放在头部区域的最前面
有序列表的项目符号有5种：1、a、A、i（小写罗马字母）、I（大写罗马字母）
无序列表的项目符号有3种：circle项目符号为空心圆、disc项目符号为实心圆、square项目符号为实心方块

属性值的控制因素：
a. 并列：值用空格隔开，必须全部出现，顺序不能改变
b. 互斥：值用|隔开，有且只能有一个值出现
c. 并且：值用&&隔开，必须全部出现，顺序可调整
d. 或者：值用||隔开，至少有一个值必须出现，顺序可调整
e. 分组符号：值用[ ]括起来，中括号之间用空格隔开，例：center和top和red就是[center|left] [top&&red]的一种属性值
f. *表示属性值出现的次数：0次、1次或多次
g. +表示属性值出现的次数：1次或多次
h. ?表示属性值可有可无
优先级：[] > 并列 > 并且&& > 或者|| > 互斥|
